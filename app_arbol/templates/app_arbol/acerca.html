🌱 Gestor de Rutas

v1.0.0

Aplicacion para organizar rutas jerarquicas de atencion o gestion dentro de una institucion. Permite modelar diferentes contextos mediante un arbol generico con nodos padre e hijo.

💡 Funcionalidades

  • Arbol generico con niveles jerarquicos
  • Multiples contextos de atencion
  • Visualizacion de arbol completo
  • Navegacion por ruta de ancestros
  • Busqueda de nodos por nombre o descripcion
  • Creacion, edicion y eliminacion de nodos

📷 Tecnologia

  • Django 4.2 (Python)
  • SQLite como base de datos
  • HTML5 + CSS3 minimalista
  • Modelo MVC (Modelo-Vista-Controlador)

🏨 Contextos incluidos

  • • Soporte tecnico
  • • Tramites administrativos
  • • Reclamos
  • • Atencion academica
  • • Orientacion al estudiante
  • • Gestion de areas internas
  • • Clasificacion de servicios

💠 Logica de arbol generico

La aplicacion implementa un arbol generico donde cada nodo puede tener multiples hijos. Las operaciones incluyen:

  • • obtener_hijos — Hijos directos de un nodo
  • • obtener_ancestros — Ruta desde la raiz al nodo
  • • obtener_descendientes — Todos los nodos debajo de uno dado
  • • construir_arbol — Arbol completo de un contexto
  • • es_hoja — Verifica si un nodo es terminal
